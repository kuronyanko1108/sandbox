# tkinter-calculator

## アプリ概要

四則演算に対応したシンプルな電卓アプリです。

以下の機能を実装しています。

- 四則演算（+ / - / \* /）
- 小数計算
- パーセント計算（%）
- 符号反転（+/-）
- AC（全クリア）

GUIには tkinter を使用しています。

## 工夫した点

### 1. 逆ポーランド記法を利用した計算ロジック

計算処理では、入力された数式を逆ポーランド記法（RPN）へ変換して計算しています。

これにより、

- 演算子の優先順位を考慮した計算
- 計算ロジックの理解
- スタックを利用したアルゴリズム学習

を目的として実装しました。

### 2. Decimalによる計算誤差対策

小数計算には `decimal.Decimal` を使用しています。

浮動小数点数特有の誤差を避け、電卓として自然な計算結果を表示できるようにしました。

### 3. 入力時の例外ケースを考慮

ユーザー操作による不正な入力を防ぐため、以下のケースを考慮しています。

- 小数点の連続入力防止
- 演算子の連続入力防止
- パーセント入力時の制御
- 不正な符号反転の防止

単に計算できるだけでなく、実際に利用することを想定して入力制御を実装しました。
